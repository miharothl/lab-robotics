# Multi-Dimensional Kalman Filter

## Prerequisites

  * [CLion](https://www.jetbrains.com/clion) (optional)
  * [CMake](https://cmake.org/)
  * [Eigen](https://eigen.tuxfamily.org/dox/GettingStarted.html)

## Installation

### Mac
```shell
brew install cmake
```

## Use

```shell
mkdir build && cd build
cmake .. && cmake
./lab_03_1d_kalman
```
